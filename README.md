## Overview

This project is a modern, responsive website designed to present Chef Zachary Richards’ professional profile and services.

The website highlights:

- Personal branding & presentation
- Professional experience
- Culinary specialization
- Contact & booking details

The design focuses on elegance, minimalism, and a premium fine-dining aesthetic.

---

## Features

- Fully responsive layout (mobile + desktop)
- Video hero section for visual impact
- Smooth scrolling navigation
- Clean typography and modern styling
- SEO & social preview metadata (Open Graph)
- Lightweight and fast-loading structure
